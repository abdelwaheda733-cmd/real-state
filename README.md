✅ Project Description (Professional English Version)

Real Estate Sales Data Analysis Project

In this project, I analyzed real estate company data using Python and built an interactive dashboard using Visual Studio Code and the Streamlit library.

The main objective of the project was to analyze sales data to support data-driven business decisions.

🔹 Project Steps:
Monthly data was stored across multiple sheets, so I first combined the datasets using joins, followed by data cleaning.
Project-related data was cleaned and prepared for analysis.
Customer data was also cleaned and standardized.
All datasets were merged into a unified dataset, which was then used for analysis.
🔹 Dashboard Development:

After completing the data analysis, I built a professional and interactive dashboard using Streamlit in Visual Studio Code.

🔹 Dashboard Overview:

Key Performance Indicators (KPIs):
At the top of the dashboard, three main KPIs provide a quick overview of business performance:

Total Revenue
Total Deals
Average Deal Size

Sales Analysis by City:
This chart shows total sales by city, helping identify top-performing locations and enabling easy comparisons.

Agent Performance Analysis:
Using a bubble chart:

Bubble size represents the number of deals
Position represents revenue

This highlights the relationship between deals and revenue and helps identify the most efficient agents.

Projects Analysis:
Displays total sales per project, helping identify the most successful projects in the market.

Customer Segmentation Analysis:
Customers are segmented into:

New
Normal
VIP

This analysis shows each segment’s contribution to total revenue, supporting better marketing strategies.

Payment Plans Analysis:
This chart shows the distribution of customers based on payment methods (cash vs. installments), helping understand customer preferences.

Geographical Analysis (Governorates):
Shows total sales by region, helping identify the best areas for investment.

Top Projects by Demand:
Highlights the most frequently sold projects, indicating high-demand and successful developments.

Property & Unit Type Analysis:

Property type (Residential / Commercial / Administrative)
Unit type (Apartment / Villa / Studio)

This helps in understanding market trends and customer needs.
